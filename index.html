Rails 3.0 and 3.1 Example Apps and Tutorials

This is a website and GitHub repository for example Rails 3.0 and 3.1 apps and tutorials.

It spans the gap between Rails tutorials for beginners and many Rails resources that assume expert knowledge.

You can use what’s here in several ways:

  • follow the tutorials and examine the code to to see examples of complete working Rails web applications
  • download code (“clone a repository”) to customize an app for your own use
  • use an application template to generate a ready-made “starter app”
  • modify an application template for your own custom starter app

Recommended Reading

The tutorials document each step that you must follow to create an example application. Every step is documented concisely, so a complete beginner can create this application without any additional knowledge. However, no explanation is offered for any of the steps, so if you are a beginner, you’re advised to look for an introduction to Rails elsewhere. See a list of recommended Rails Books.
